索引的压缩​

一些简单的编码​

  • Unary Code
    • 对于每一个整数 x，x 编码为 (x - 1) 个 1，以 0 结束
  • Elias-γ Code
    • 对于一个整数 x(x > 0)，由两部分组成
      • N=[log2x]N=[\text{log}_2x]N=[log2​x]，用 N 个 1 来表示
      • K=x−2[log2x]K=x-2^{[\text{log}_2x]}K=x−2[log2​x]，K 用二进制编码，长度为 N，中间用 0 分割
    • 需要1+2[log2x]1+2[\text{log}_2x]1+2[log2​x]个二进制位
  • Elias-𝛿 Code
    • 与 Elias-γ Code 过程类似，只是将 N+1 按 Elias-γ Code 再一次分解
    • 需要1+2[log2log22x]+[log2x]1+2[\text{log}_2\text{log}_22x]+[\text{log}_2x]1+2[log2​log2​2x]+[log2​x]个二进制位

求数字 9 的个压缩码：

  • Unary Code：11111111 0
  • Elias-γ Code：111 0 001
  • Elias-𝛿 Code：11 0 00 001

Golomb Code​

Golomb编码是一种基于游程编码（run-length encoding，RLE）的无损的数据编码方式，当待压缩的数据符合几何分布（Geometric Distribution）时，Golomb编码取得最优效果。

给定数值b (b=ln(2)×Avg)b\, (b=\text{ln}(2)\times \text{Avg})b(b=ln(2)×Avg)，Avg\text{Avg}Avg为编码数值的平均数

取

  • 商：q=[(x−1)/b]q=[(x-1)/b]q=[(x−1)/b]
  • 余数：r=(x−1)−q∗br=(x-1)-q*br=(x−1)−q∗b
  • c=[log2b]c=[\text{log}_2b]c=[log2​b]

对xxx编码，将下面两部分连接起来

  • 对q+1q+1q+1采用 Unary 编码
  • 对rrr用ccc或c+1c+1c+1位二进制进行编码
    • 如果r<2c−1r<2^{c-1}r<2c−1，则使用ccc位二进制位，以 0 开始
    • 否则使用c+1c+1c+1个二进制位，第一位为 1，剩余部分为r−2c−1r-2^{c-1}r−2c−1二进制编码

举例：给出数字 5 的 Golomb Code，b = 3

  • q=[(5−1)/3]=1q=[(5-1)/3]=1q=[(5−1)/3]=1
    • q+1:10q+1:10q+1:10
  • r=(x−1)−q∗b=4−(1∗3)=1r=(x-1)-q*b=4-(1*3)=1r=(x−1)−q∗b=4−(1∗3)=1，用 2 位编码
  • 最终编码为：1010

签名文档​

签名文档将文本切分成若干个块，每个块中包含bbb个词项，将单词映射到BBB位的位掩码，则文档的签名通过对块中所有词的掩码按位或操作得到。

查询​

给定一个qqq，产生查询式的签名SqS_qSq​，则匹配条件为Sq∪Sb=SqS_q\cup S_b=S_qSq​∪Sb​=Sq​

不同词的签名重叠，很可能导致查询到的文本中并不包含待查询词，所以签名文档的最好是用来剔除那些不含查询词项的文档。
